# web-komunitas
Web Komunitas ini dibuat sebagai ruang berkumpulnya anggota, pegiat, dan masyarakat yang ingin terhubung, berbagi informasi, serta mengembangkan kegiatan bersama. Melalui platform ini, setiap pengguna dapat mengikuti event terbaru, membaca artikel inspiratif, berbagi pengalaman, serta berkolaborasi dalam berbagai program komunitas.
